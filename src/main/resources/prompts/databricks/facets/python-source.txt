=== RULES ===

Add rules covering:
- **Schemas are declared, never inferred in production paths**: production reads use `spark.read.schema(SCHEMA).format(...)`; inference is only acceptable in exploratory notebooks.
- **No `.collect()` on production frames**: aggregate down to a known-bounded size first, or write directly; collecting an unbounded frame to the driver is a memory-exhaustion bug waiting to happen.
- **dbutils only in notebook context**: plain library modules and tests do not import `dbutils`; pass paths and parameters in explicitly so the module is testable outside Databricks.
- **Idempotent writes**: production writes use `mode("overwrite")` with a deterministic partition key, or `MERGE INTO` with a stable merge condition; never `append` without de-dup logic that can survive a job retry.
- **Spark stops in scripts**: plain-script Python (non-notebook) explicitly calls `spark.stop()` at the end so the driver releases cluster resources.
